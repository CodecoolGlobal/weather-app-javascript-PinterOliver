## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CodecoolGlobal/weather-app-javascript-PinterOliver.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the application
   ```sh
   cd .\backend\
   ```
   ```sh
   node server.js
   ```
